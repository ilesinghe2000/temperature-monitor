01 — Overview

Project Goal

The goal of this project is to build a small IoT monitoring device using an Arduino Uno microcontroller. The system continuously reads temperature (°C) and relative humidity (%) from a DHT11 sensor and displays the values on a 16×2 LCD screen in real time.

This project demonstrates core embedded systems concepts: sensor interfacing, I2C communication, real-time data display, and serial logging — making it a practical, fully functional IoT device.

🌡️

Real-time sensing

Reads temperature and humidity every 2 seconds using the DHT11 sensor.

🖥️

LCD display

Shows live readings on a 16×2 LCD connected via I2C (only 2 wires needed).

📡

Serial logging

Streams data to the PC via USB for monitoring and debugging in real time.

🔥

Heat index

Calculates the "feels like" temperature when conditions are hot and humid.

03 — Wiring

Circuit Connections

The DHT11 uses a single digital data line. The LCD uses I2C, which only needs 2 signal wires (SDA and SCL), greatly reducing wiring complexity. 

── DHT11 Sensor ──────────────────────────────
DHT11 VCC   →  Arduino  5V
DHT11 GND   →  Arduino  GND
DHT11 DATA  →  Arduino  Pin 2  +  10kΩ resistor to 5V

── LCD 16×2 (I2C module) ─────────────────────
LCD VCC     →  Arduino  5V
LCD GND     →  Arduino  GND
LCD SDA     →  Arduino  A4   (I2C data)
LCD SCL     →  Arduino  A5   (I2C clock)

── Power ──────────────────────────────────────
Arduino powered via USB from PC
04 — Code

Arduino Sketch

Written in C++ using the Arduino framework. Three libraries are used: DHT for the sensor, Wire for I2C, and LiquidCrystal_I2C for the display.

temperature_monitor.ino Arduino C++ 
// Libraries
#include <DHT.h>
#include <Wire.h>
#include <LiquidCrystal_I2C.h>

// Pin config
#define DHTPIN  2
#define DHTTYPE DHT11

DHT dht(DHTPIN, DHTTYPE);
LiquidCrystal_I2C lcd(0x27, 16, 2);

void setup() {
  Serial.begin(9600);
  dht.begin();
  lcd.init();
  lcd.backlight();
}

void loop() {
  delay(2000);
  float humidity = dht.readHumidity();
  float tempC    = dht.readTemperature();

  if (isnan(humidity) || isnan(tempC)) {
    lcd.print("Sensor Error!");
    return;
  }

  lcd.setCursor(0, 0);
  lcd.print("Temp: "); lcd.print(tempC);  lcd.print("C");
  lcd.setCursor(0, 1);
  lcd.print("Humi: "); lcd.print(humidity); lcd.print("%");
}
05 — How it Works

System Flow

The system runs a continuous loop, reading and displaying data every 2 seconds.

1

Power on & initialize

Arduino starts, initializes the DHT11 sensor and LCD display. A splash screen shows for 2 seconds.

2

Read sensor data

Every 2 seconds, the DHT11 measures ambient temperature (°C) and relative humidity (%). The data is sent over a single digital wire to pin 2.

3

Validate the reading

The code checks if the values are valid numbers. If the sensor is disconnected or faulty, an error message is shown instead of garbage data.

4

Display on LCD

Valid readings are written to the 16×2 LCD via I2C. Row 1 shows temperature, Row 2 shows humidity. The display updates every cycle.

5

Log to serial monitor

The same data is also printed to the Arduino IDE Serial Monitor over USB, useful for debugging and recording data over time.
